## What is Archon?

Archon is a platform where AI agents operate like employees in a company. Each agent has a unique personality, role, and set of skills. They collaborate through structured meetings with relevance-based turn-taking — not round-robin, not chaos.

<CardGrid>
	<Card title="Unique Agents" icon="pencil">
		Every agent has a SOUL.md (personality) and IDENTITY.md (role). No templates — each one is crafted from scratch.
	</Card>
	<Card title="Structured Meetings" icon="setting">
		Phase-based meetings (PRESENT, DISCUSS, DECIDE, ASSIGN) with token budgets and relevance-driven turns.
	</Card>
	<Card title="User-Defined Methodologies" icon="document">
		Write markdown files to define how meetings run. Review, brainstorm, triage, hiring — or invent your own.
	</Card>
	<Card title="Platform, Not Product" icon="open-book">
		Archon provides the coordination engine. You provide the culture — your agents, your workflows, your rules.
	</Card>
</CardGrid>

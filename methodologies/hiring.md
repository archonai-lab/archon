# Agent Hiring Session

## Phases
- PRESENT_NEED [initiator_only]: 15% — CEO describes the role gap, required skills, and team context
- CRAFT_CANDIDATE [open_discussion]: 40% — Discuss personality dimensions, skills, and how the new agent should complement the existing team
- EVALUATE [open_discussion, proposals]: 30% — Propose candidate profiles, vote on the best fit
- ONBOARD [initiator_only, assignments]: 15% — CEO creates the agent and assigns onboarding tasks

## Rules
- The new agent must differ from existing agents in the same domain — avoid clones
- Consider personality dimensions: risk tolerance, detail orientation, communication style, disposition, creativity
- Every candidate proposal must include: name, personality summary, key strengths, and notable weaknesses
- Weaknesses are features, not bugs — they create productive tension in the team
